# **MyBike Landing Page**

A modern, responsive landing page for a bike brand, focused on clean UI, clear content structure, and smooth user interaction.
The project demonstrates strong fundamentals in semantic markup, SCSS architecture, and vanilla JavaScript without using frontend frameworks.

---

## **LIVE DEMO**

**Experience the live website:**
👉 [https://github.com/iirk1/MyBike-Landing-Page]

---

## **Technologies**

- **HTML5** — semantic, accessible markup
- **SCSS (Sass)** — modular, well-structured styles with variables and nesting
- **Vanilla JavaScript** — UI interactions and basic logic
- **Google Fonts** — custom typography
- **Git & GitHub Pages** — version control and deployment

## **Installation**

### **Clone the repository**

```bash
git clone https://github.com/your-username/project-name.git
cd project-name
```

### **Install dependencies:**

```bash
npm install
# or
yarn install
```

### **Run the project locally:**

```bash
npm start
# or
yarn start
```

## **Features**

- **Fully responsive layout** (mobile / tablet / desktop)
- **Burger menu** with overlay navigation
- **Smooth section-based layout** with clear content hierarchy
- **Product comparison section** with reusable components
- **Interactive contact form** with client-side reset handling
- **Clean, maintainable codebase** without frontend frameworks
- **Performance- and UX-oriented implementation**
